# 设计系统草案

## 1. 设计目标
为 `3D Science Lab` 提供一套统一的视觉语言，让产品同时具备：
- 教育产品的可信度
- 科技产品的未来感
- 教学工具的清晰度
- 多角色页面的一致性

## 2. 风格定位
- 风格关键词：`未来感`、`可信`、`清晰`、`教育科技`
- 视觉方向：深色科技底 + 高亮蓝绿强调色 + 玻璃拟态卡片
- 页面原则：信息层次清晰、主 CTA 明确、避免堆砌炫技特效

## 3. 色彩系统

### 主色
- `Primary / 500`：`#5B8CFF`
- `Primary / 600`：`#4A79F0`
- `Primary / 700`：`#345FCA`

### 强调色
- `Accent / Mint`：`#38E0C1`
- `Accent / Cyan`：`#65D9FF`

### 中性色
- `Background / 900`：`#07111F`
- `Background / 800`：`#0D1830`
- `Surface / 700`：`#101F3A`
- `Border`：`rgba(133, 163, 255, 0.22)`
- `Text / Primary`：`#F4F8FF`
- `Text / Secondary`：`#9CB0D2`

### 语义色
- `Success`：`#2ED47A`
- `Warning`：`#FFB648`
- `Error`：`#FF6B7A`

## 4. 字体系统
- 中文优先：`PingFang SC`, `Microsoft YaHei`, `Inter`, `sans-serif`
- 标题：更紧凑，字重 `700-800`
- 正文：字重 `400-500`
- 行高：正文保持 `1.6 - 1.75`

### 建议字号
- Hero 标题：`48-72px`
- 页面标题：`32-40px`
- 模块标题：`24-28px`
- 正文：`16-18px`
- 辅助文案：`14px`

## 5. 间距系统
- 基础单位：`4px`
- 常用间距：`8 / 12 / 16 / 24 / 32 / 48 / 64`
- 页面区块上下留白：`72 - 96px`
- 卡片内边距：`20 - 28px`

## 6. 组件规则

### 按钮
- 最小点击高度：`44px`
- 主按钮：蓝色渐变背景
- 次按钮：透明/描边
- Hover：仅做轻微位移和阴影增强

### 卡片
- 圆角：`16 - 24px`
- 背景：半透明深色面板
- 边框：低对比发光描边
- 阴影：柔和大面积阴影，避免过黑

### 标签 / Chip
- 用于学段、学科、模式、状态
- 颜色区分但不依赖颜色单独表达含义

## 7. 页面模板

### 营销页
- Hero
- 价值说明
- 实验矩阵
- 教师/学校闭环
- CTA 试用申请

### 学生端
- 顶部导航简洁
- 首屏优先展示推荐任务
- 卡片列表强调实验目标、时长、模式

### 教师端
- 首屏优先显示摘要数据
- 默认提供操作入口，不逼老师思考太多
- 错误分析以“教学建议”形式输出

## 8. 可访问性规则
- 所有交互元素高度不低于 `44px`
- 文本对比度不低于 `4.5:1`
- 键盘聚焦必须可见
- 图标按钮必须补 `aria-label`
- 表格信息要有文本表头，不只靠颜色

## 9. 动效建议
- 按钮 / 卡片：`150-220ms`
- 页面过渡：不超过 `300ms`
- 优先使用 `transform` 和 `opacity`
- 支持 `prefers-reduced-motion`

## 10. 当前适用范围
- `frontend/showcase/`：高保真静态展示页
- `frontend/webapp/`：正式 React 前端壳
