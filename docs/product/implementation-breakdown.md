# 产品实现拆解

## 1. 目标
把当前产品文档进一步落到设计、前端、3D、后端可以并行推进的任务层。

## 2. 并行工作流

### 线 A：页面与交互原型
- 先完成 `P0` 页面低保真原型
- 再把页面转成高保真 UI
- 再接入真实数据和实验配置

### 线 B：实验配置与引擎
- 先定义统一 `Experiment Schema`
- 再录入首批 6 个实验配置
- 再建设共享交互能力层

## 3. 页面任务拆解

### 学生端
- 登录页：角色切换、演示账号入口、基础错误提示
- 实验大厅：今日任务、筛选、实验卡片、状态标签
- 实验详情页：目标说明、器材预览、模式选择、开始按钮
- 3D 实验页：步骤栏、场景区、反馈区、底部主操作栏
- 结果页：得分、错因、复盘建议、再练一次

### 教师端
- 工作台：班级概览、最近布置、快捷操作、课堂演示入口
- 班级实验详情页：完成率、平均分、错误步骤、学生列表

## 4. 共享引擎任务拆解

### P0 共性能力
- 实验配置加载器
- 步骤状态机
- 对象高亮系统
- 拖拽 / 放置系统
- 连线系统
- 材料添加系统
- 观察记录面板
- 结果评分引擎
- 结果页数据汇总

### P1 增强能力
- 微观视角切换
- 仪表读数动画
- 数据表记录
- 危险动作拦截
- 条件变量控制

## 5. 6 个实验的研发任务

### `phy-junior-circuit-001`
- 场景：实验台、电池盒、灯泡、开关、导线
- 交互：导线吸附连接、闭合回路判定、灯泡亮灭反馈
- 结果：串联/并联差异总结

### `chem-junior-oxygen-001`
- 场景：试管、导管、水槽、集气瓶、酒精灯
- 交互：装置搭建、药品加入、加热触发、气泡收集
- 风险：危险步骤拦截与规范提示

### `bio-junior-microscope-001`
- 场景：显微镜、载玻片、样本
- 交互：对光、样本放置、粗调/细调、视野变化
- 结果：清晰度与观察记录输出

### `phy-senior-ohm-001`
- 场景：电源、电阻、电流表、电压表、变阻器
- 交互：复杂接线、电表极性与连接校验、读数记录
- 结果：数据趋势和实验结论

### `chem-senior-galvanic-001`
- 场景：双烧杯、电极、盐桥、电流计
- 交互：装置完整性校验、宏观现象、微观视角切换
- 结果：原电池形成条件总结

### `bio-senior-enzyme-001`
- 场景：试管组、条件控制模块、计时器
- 交互：设置对照组、切换变量、记录结果
- 结果：变量控制与结论准确度评分
