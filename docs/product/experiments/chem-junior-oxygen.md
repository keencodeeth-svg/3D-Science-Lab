# 初中化学：氧气制备与收集

## 1. 基本信息
- 实验 ID：`chem-junior-oxygen-001`
- 学段：初中
- 学科：化学
- 建议年级：九年级
- 推荐模式：引导 / 练习
- 目标时长：13 分钟

## 2. 教学目标
- 理解氧气制备与收集的基本流程
- 认识试管、导管、集气瓶、酒精灯等器材
- 观察气体生成与收集过程中的关键现象

## 3. 线下痛点与线上价值
- 线下涉及加热和玻璃器材，存在安全风险
- 装置安装顺序容易出错，学生复练成本高
- 在线可拦截危险步骤，并突出装置连接关系

## 4. 3D 场景与器材
- 试管
- 铁架台
- 酒精灯
- 导管
- 水槽
- 集气瓶
- 药品容器

## 5. 核心交互步骤
1. 选择并摆放试管、铁架台和导管
2. 加入指定药品并固定试管
3. 检查装置气密性
4. 点燃酒精灯并开始加热
5. 观察气泡生成并收集气体
6. 判断收集完成时机
7. 完成验满与收尾

## 6. 判定与评分
- 装置连接是否正确
- 加热顺序是否规范
- 是否完成气密性检查
- 收集时机是否正确
- 评分建议：规范操作 60% + 现象判断 40%

## 7. 常见错误提示
- 导管连接错误或未插紧
- 未先检查气密性就开始加热
- 酒精灯使用顺序错误
- 集气瓶位置不正确导致收集失败

## 8. 结果页输出
- 装置搭建正确率
- 操作规范得分
- 危险操作提醒次数
- 核心结论：氧气制备与收集的关键注意事项

## 9. 通用能力复用
- 器材摆放
- 加热状态变化
- 气泡/气体动画
- 危险动作拦截
