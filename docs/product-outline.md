# 产品初步定义

## 产品定位

一个专注初中、高中理科实验教学的三维在线模拟实验平台，支持物理、化学、生物三大学科，以“可视化、可交互、可评估”为核心。

## 核心价值

- 降低实验器材和场地限制
- 提升危险实验的安全性
- 支持课前预习、课堂演示、课后复习
- 为教师提供可管理、可追踪的教学工具

## MVP 建议范围

### 物理
- 电路连接与测量
- 杠杆与力学平衡
- 光学成像

### 化学
- 酸碱中和
- 气体制备与收集
- 常见反应现象观察

### 生物
- 细胞结构观察
- 显微镜使用
- 光合作用/呼吸作用过程演示

## 关键模块

- 3D 实验场景
- 实验器材交互系统
- 步骤引导与任务判定
- 错误提示与实验结果反馈
- 教师端实验编排与课堂管理
- 用户登录、进度记录与数据看板
