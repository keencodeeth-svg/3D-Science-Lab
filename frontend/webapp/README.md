# WebApp Shell

这是正式前端壳的起点，目标是：
- 用 `React + TypeScript + Vite` 承载学生端和教师端
- 从 `backend/configs/experiments/` 同步实验配置到前端静态数据目录
- 运行时通过 `/api/v1/experiments` 从后端读取实验目录与配置
- 通过统一课程库支撑小学科学 + 初高中理化生实验扩容
- 通过按需加载和分包策略承载后续更多 3D 实验场景

## 目录
- `src/`：前端页面与组件
- `public/data/experiments/`：同步后的实验数据构建产物
- `scripts/sync-experiments.mjs`：同步脚本

## 推荐命令
- `npm install`
- `cd ../../backend && npm run dev`
- `npm run sync:data`
- `npm run verify:data`
- `npm run dev`
- `npm run build`

开发模式下，前端会通过 Vite 代理把 `/api` 转发到 `http://127.0.0.1:4318`，实验目录与实验配置都由后端 API 提供。
